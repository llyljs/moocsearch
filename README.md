# Moocsearch
A web app that download mooc video and lecture materials using Flask framework

### Requirements:
This project utilizes the following requirements:
1. python v3.6.5
2. flask v1.0
3. lxml v4.1.1
4. beautifulsoup4 v4.6.0
5. requests v2.18.4
